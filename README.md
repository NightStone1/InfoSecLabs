# InfoSecLabs

Лабораторные работы по информационной безопасности, реализованные на C# и .NET.

## Структура
- InfoSec.Common — общие утилиты и расчёт параметров варианта
- Lab1 — reuse of keystream / XOR
- Lab2 — Caesar и Vigenere
- Lab3 — PRNG vs CSPRNG
- Lab4 — SHA-256, HMAC, PBKDF2
- Lab5 — AES-256-GCM
- Lab6 — AES-CBC + HMAC

## Запуск
Открыть InfoSecLabs.sln в Visual Studio и выбрать нужный проект как Startup Project.
